

## 替换命令


```
回车生效


公式
:[range] s [ubstitute] / {pattern} / {string} / [flags]

rang: 
	$ 到尾部（用在number,number里的）
	% 全文
	number,number [第x行到第y行]
	默认替换匹配到的第一个

pattern:
	正则

flags
	g 全局
	c 询问是否进行修改


例子：
1. 全局替换vnode -> haha
:%s/vnode/haha






```



```

visual模式下，选中内容

按下冒号后

:'<,'>


'< visual选中区域的开头
>' visual选中区域的结尾

也是number，number的语法

```
